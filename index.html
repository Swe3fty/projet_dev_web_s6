Page d'accueil


Projet de Développement Web réalisé dans le cadre de nos études à l’Isen Brest. Il a pour but de visualiser les données concernant les bornes de recharge des véhicules électriques (issues de la base nationale IRVE). Le but était de concevoir et développer une application d’étude des données IRVE et d'approfondir les compétences acquises dans les modules de Big Data, d'Intelligence Artificielle, de Web et de Base de données.
